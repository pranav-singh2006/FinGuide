# FinGuide

FinGuide is a modern, responsive personal budget planner designed to help you balance your income by automatically calculating rule-based allocations effortlessly.

## Objective
Help users allocate their monthly net income into three categories based on the widely used 50/30/20 rule (default):
- **Needs (Fixed Expenses):** 50%
- **Wants (Lifestyle/Flex):** 30%
- **Savings/Debt:** 20%

## Features
- **Modern "FinTech" Aesthetic:** Clean interface using Tailwind CSS, featuring "Success Green" for savings and "Royal Blue" for primarily actions.
- **Dynamic 50/30/20 Engine:** Automatically calculate suggested budgets based on your exact net income. Don't like 50/30/20? The ratios are fully dynamic and adjustable!
- **Interactive Visual Analytics:** Features a high-quality React-Chartjs-2 Doughnut chart showing the precise breakdown of your allocations.
- **Financial Health Insights:** Alerts you immediately if your actual fixed expenses exceed your targeted needs ratio.
- **Responsive & Accessible (WCAG):** Fully supportive of all device screens.

## Technical Stack
- **Framework:** React 19 (via Vite)
- **Styling:** Tailwind CSS (v4)
- **Charts:** Chart.js + react-chartjs-2
- **Icons:** Lucide React

## Local Setup
1. Ensure Node.js is installed.
2. Clone or download this repository.
3. Open your terminal in the target directory.
4. Run `npm install` to install dependencies.
5. Run `npm run dev` to start the local development server.

## Deployment to GitHub
This project is built using a simple, modular single-page architecture format, making it extremely easy to upload to GitHub and deploy on services like Vercel or GitHub Pages.
